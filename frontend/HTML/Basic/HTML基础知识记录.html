什么是HTML #

HTML 是用来描述网页的一种语言

  • HTML指的是超文本标记语言(Hyper Text Markup Language)
  • HTML不是一种编程语言而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

文档声明和字符编码 #

!DOCTYPE一个文档类型标记是一种标准通用标记语言的文档类型声明，它的目的是要告诉标准通用标记语言解析器，它应该使用什么样的文档类型定义（DTD）来解析文档。

<!DOCTYPE html> 代表当前html5标准，特殊且固定的文档声明

<html lang="***">告诉浏览器当前采用的是什么语言

<meta charset="***">告诉浏览器当前什么编码格式，浏览器会以对应编码格式解码显示内容。如果编码格式和实际编码不一致，显示内容则会乱码
